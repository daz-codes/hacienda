# Solanum branding (parked)

Visual identity explored for a future **Solanum** site — parked here, not used
by the store example (which keeps its Haçienda / FAC 51 industrial theme).

The name plays on **PORO** (Plain Old Ruby Objects) → *poroporo*, the New
Zealand nightshade, genus **Solanum**.

## Assets

- `solanum.svg` — logo: a five-petal violet nightshade flower with a solid
  pollen-yellow centre. Petals are evenly spaced (no overlap). 64×64 viewBox.
- `solanum.css` — botanical theme: white ground, violet (`#7c3aed`) UI accents,
  foliage-green (`#2f7a4b`) headings, pollen-yellow flower centre.

## Fonts

The stylesheet expects these web fonts (load via Google Fonts or self-host):

```
Fraunces  — display / headings (sharp botanical serif)
Oswald    — nav, buttons, labels (condensed grotesque)
Archivo   — body text
```

## Notes / open questions

- The frontal five-petal flower is intentionally close to the **Hanami** logo
  style. A profile/pendant nightshade silhouette (petals swept back, yellow
  anther cone pointing down) was also explored as a more distinct alternative.
- Command shorthand idea: `sol` (mirrors the Hacienda `fac`), not yet applied.
